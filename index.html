Flex Container Properties

  • display-: flex -> inline-flex
  • flex-direction-: row -> row-reverse -> column -> column-reverse
  • flex-wrap-: nowrap(default) -> wrap -> wrap-reverse
  • flex-flow (Combination of flex-direction and flex-wrap)
  • justify-content(main axis)-: flex-start,flex-end,center,space-between,space-around,space-evenly
  • align-content cross axis only work when multiple row in the conatiner :- stretch(default) -> flex-start - flex-end - center -> space-between -> space-around
  • align-items(cross axis)
